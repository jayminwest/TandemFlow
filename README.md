# TandemFlow Landing Page

A modern, responsive landing page for TandemFlow - a business automation agency. Built with React, Vite, and TailwindCSS.

## Features

- Modern, responsive design
- Smooth animations and transitions
- Interactive components
- Typeform integration for lead capture
- Dark theme with teal accents
- Mobile-friendly navigation
- Case studies showcase
- Pricing plans
- FAQ section
- Legal and Privacy pages

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router DOM
- React Icons
- Typeform Embed React

## Project Structure

The project follows a standard React application structure with components, pages, and shared utilities:

- `/src/components`: Reusable UI components
- `/src/pages`: Page-level components
- `/src/assets`: Static assets
- `/public`: Public assets

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`

## Key Components

- Hero Section: Dynamic hero section with animated background
- Services: Showcases automation services with hover effects
- Case Studies: Displays successful client implementations
- How We Work: Step-by-step process explanation
- Pricing: Tiered pricing plans with interactive cards
- FAQ: Expandable FAQ section
- Footer: Comprehensive footer with newsletter signup

## Styling

The project uses TailwindCSS for styling with custom configurations for:
- Colors

- Responsive design
